### Development

Starts up a development server, which will automatically rebuild the app as you change files and supports hot module replacement for fast development.

```
npm run dev
```

### Build/Production

Builds the assets once

```
npm run build
```

### Serve

Starts the server for `dist`

```
npm run serve
```
